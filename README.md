# Fitness-app
"Antrenament Interactiv - Fitness" este o aplicație dedicată celor pasionați de fitness, oferind antrenamente personalizate și interacțiuni dinamice. Aceasta permite utilizatorilor să introducă datele personale precum numele, înălțimea, greutatea, vârsta și sexul, pentru a personaliza rutina de antrenament în funcție de caracteristicile fizice.
